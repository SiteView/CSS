AuthorizeId

'Authorize Id' ist ein Plugin für CSS, welches mit LDAP-Einträgen arbeitet, die in der Gruppe EpicsAuthorizeID sind. Dies Plugin kann Einträge darstellen, neue Einträge erzeugen, Einträge editieren und löschen. Einträge in der ersten Tabelle sind Namen, in der zweiten Tabelle sind Attribute dieser Namen. Wird ein Name editiert oder gelöscht, werden auch alle Attribute automatisch editiert oder gelöscht.

If entry name is not valid?

Das Plugin verhindert, dass ungültige Namen eingegeben werden; z.B. leere Namen oder Namen mit von LDAP nicht unterstützten Zeichen.


Tips and tricks

Um Feldnamen oder Attribute schnell zu finden, kann man sie einfach eintippen; es wird automatisch auf die Zeile gesprungen, wo sie vorkommen. Main kann auch auf Spaltenköpfe klicken, um die Namen oder Attribute alfabetisch zu sortieren.

Der Text, der auf dem Bildschirm erscheint, kann in einer Datei namens "messages.properties" geändert werden.
Die Timeout-Vorgaben für LDAP-Datenzugriffe können in Klassen : LdapEain, LdapGroups und LdapProp geändert werden, wenn sie zu groß oder zu klein sind.

BUGS:

---
Ein Fehler passiert unter speziellen Umständen, beispielsweise, wenn CSS einige Zeit nicht läuft (PC ist aus) und wenn man CSS neu startet und AuthorizeId zum ersten Mal startet, dann können Gruppen leer sein.

Der Grund für diesen Fehler ist unklar, aber der Author nimm an, das es beim erstmaligen Verbinden mit LDAP ein Timing-Problem gibt.

Schließe und wieder Öfnnen des Plugins beseitigt das Problem. Man muß CSS nicht neu starten.
---

Bitte beachten : der Fehler ist beseitigt, aber um sicher zu sein, hat der Author diese Fehlerbeschreibung bestehen lassen, falls es nochmal ähnliche Probleme gibt.
