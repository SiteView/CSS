AlarmViewer
Beschreibung Zeigt eine Liste der aktuellen Alarmnachrichten an.
Optionen topics = Topic1[,Topic2,...]
Zeigt nur die Nachrichten in den betreffenden Topics an.
cmd = list
Zeigt eine Liste aller möglichen Topics an.
Link http://cssweb.desy.de:8085/AlarmViewer

AlarmViewerXml

Optionen topics = Topic1[,Topic2,...]
Zeigt nur die Nachrichten in den betreffenden Topics an.

Zeigt eine Liste aller möglichen Topics an.
