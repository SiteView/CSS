Description of the TestAllChannelCombinations
Need Modules (M) and every
 - PureChannel as Input and Output (PCI/PCO)
 - StructChannel as Input and Output (SCI/SCO)
combination.

Possible combinations: 
M->PCI PCI->SCI
M->PCO SCI->PCI
M->SCI PCO->SCO
M->SCO SCO->PCO
The test  sequence:
M1-->PCI11(INT16)-->SCO11(INT8 )-->PCO11(UINT16)-->SCI11(UINT8 )-->SCO12(UNIT16)-->PCI12(INT8)
M2-->SCI21(INT8 )-->PCO21(INT16)-->PCI21(UINT8 )-->SCO21(UINT16)-->PCO22(INT8)-->SCI22(UINT16)
