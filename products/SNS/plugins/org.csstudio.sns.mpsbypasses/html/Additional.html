Additional MPS Bypass Display Options

The CSS MPS Bypass Tracker enables users to see and do more than view MPS Bypass Status. Users can send Bypass Information to the E-log and view Web pages from the Bypass Request System.

Elog Information

To send the information being displayed to the E-log, press the Bypass Info to Elog button. Pressing this button will display the dialog below.

The Logbook option is a pulldown menu containing all the possible logbook selections. The title and text entries are written for the user, but both areas are editable. So information can be added or removed from these text entry areas. To send the Elog, the user must enter their UCAMS UID and password. Once all the data is acceptable, the user can press the OK button to send it to the Elog or the Cancel button to cancel the Elog entry process.

Bypass Request Web System

There are 4 selections under the Bypass Request Web Options. To see and select options, press the Press for Web Page Options button. The options are listed and explained below.

SNS Bypass Request Web Page Options

The SNS Bypass Request System has several options, but only 4 of these options are presented here.
  1. InputBypass

    2. Displays the web page, shown below, for Requesting a Bypass.

    Bypass Request Page

  2. ViewInstalledBypass

    3. Displays the web page, shown below, for Viewing Installed Bypasses.

    View Page of Bypasses

  3. ViewPreviouslyRemovedBypass

    View Previously Removed Bypasses

  4. ViewRemovedBypass
