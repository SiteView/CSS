Other Information Displayed

Information explaining the Beam and Machine Modes of the RTDL and the MPS Switch is displayed. As well as information explaining the color scheme used in the display of the State and Bypass Requested? columns.

Beam and Machine Modes

Before the SNS was in operation, MPS engineers would manually switch the modes, which overrode the RTDL settings. As a result, Operations wanted confirmation that their settings were not being overwritten by MPS engineers. This information remains on the display for reassurance. MPS PVs are read to determine the setting of the RTDL and MPS Switch Beam and Machine Modes. There are 8 Beam Modes and 7 Machine Modes. A legend is displayed showing the RTDL and Switch settings for the Beam Mode and the Machine Mode. These settings should be the same.

SNS Beam Modes

The options for Beam Modes at the SNS are:
  1. 10uSec

  2. 50uSec

  3. 100uSec

  4. 1mSec

  5. FullPwr

  6. Off

  7. StandBy

  8. MPSTest

The MPS Pvs used to determine the RTDL Beam Mode are ICS_MPS:RTDL_BmMd:10uSec.RVAL, ICS_MPS:RTDL_BmMd:50uSec.RVAL, ICS_MPS:RTDL_BmMd:100uSec.RVAL, ICS_MPS:RTDL_BmMd:1mSec.RVAL, ICS_MPS:RTDL_BmMd:FullPwr.RVAL, ICS_MPS:RTDL_BmMd:Off.RVAL, ICS_MPS:RTDL_BmMd:StandBy.RVAL and ICS_MPS:RTDL_BmMd:MPSTest.RVAL. The pv's name whose value is 1.0 represents the RTDL Beam Mode setting.

The MPS Pvs used to determine the Switch Beam Mode are ICS_MPS:Switch_BmMd:10uSec.RVAL, ICS_MPS:Switch_BmMd:50uSec.RVAL, ICS_MPS:Switch_BmMd:100uSec.RVAL, ICS_MPS:Switch_BmMd:1mSec.RVAL, ICS_MPS:Switch_BmMd:FullPwr.RVAL, ICS_MPS:Switch_BmMd:Off.RVAL, ICS_MPS:Switch_BmMd:StandBy.RVAL and ICS_MPS:Switch_BmMd:MPSTest.RVAL. The pv's name whose value is 0.0 represents the Switch Beam Mode setting.

SNS Machine Modes

The options for the Machine Mode at the SNS are:
  1. MEBT_BS - Medium Energy Beam Transport Beam Stop

  2. CCL_BS - Normal Conducting Linac Beam Stop

  3. LDmp - Linac Dump

  4. IDmp - Ring Injection Dump

  5. Ring - Ring

  6. EDmp - Ring Extraction Dump

  7. Tgt - Target Systems


The MPS Pvs used to determine the RTDL Machine Mode are ICS_MPS:RTDL_MachMd:MEBT_BS.RVAL, ICS_MPS:RTDL_MachMd:CCL_BS.RVAL, ICS_MPS:RTDL_MachMd:LDmp.RVAL, ICS_MPS:RTDL_MachMd:IDmp.RVAL, ICS_MPS:RTDL_MachMd:Ring.RVAL, ICS_MPS:RTDL_MachMd:EDmp.RVAL and ICS_MPS:RTDL_MachMd:Tgt.RVAL. The pv's name whose value is 0.0 represents the RTDL Machine Mode setting.

The MPS Pvs used to determine the Switch Machine Mode are ICS_MPS:Switch_MachMd:MEBT_BS.RVAL, ICS_MPS:Switch_MachMd:CCL_BS.RVAL, ICS_MPS:Switch_MachMd:LDmp.RVAL, ICS_MPS:Switch_MachMd:IDmp.RVAL, ICS_MPS:Switch_MachMd:Ring.RVAL, ICS_MPS:Switch_MachMd:EDmp.RVAL and ICS_MPS:Switch__MachMd:Tgt.RVAL. The pv's name whose value is 1.0 represents the Switch Machine Mode setting.
