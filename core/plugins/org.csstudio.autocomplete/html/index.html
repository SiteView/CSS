Overview

The PV Auto-Complete tool helps user to find PV names displaying choices when the user starts typing.

If there is no implementation for name lookup, the user interface simply remembers names that were entered before.

If there is a name lookup implementation available, whenever a PV name need to be entered, CSS offers assistance by listing possible values. When typing "T" for instance, the 10 first PV names starting by T will be displayed with an indicator of xxx more... It allows wildcards * and ?.

Usage

Start typing and the popup will appears automatically. You can force it using ctrl + space. Some special characters are allowed:

  • * => replace 1 to n characters.
  • ? => replace only 1 character.
